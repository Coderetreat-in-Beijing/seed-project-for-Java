# TDD Kit - Java [![Travis status]](https://travis-ci.org/arpinum/tdd-kit-java)

## Run unit tests

    gradle test
    
## Run lint, unit tests and coverage

    gradle    

## Libraries documentation

* Assertion api: [assertj]
* Test runner: [junit]
* Linter: [pmd]
* Coverage: [JaCoCo]
* Mock api: [mockito]
